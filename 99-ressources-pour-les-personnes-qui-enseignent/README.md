## Avant le cours

### Email envoyé aux personnes qui étudient avant le premier cours

```text
[DévProdMéd 2024-2025] Informations générales sur le cours

Bonjour tout le monde,

J'espère que vous allez bien et que votre semestre a commencé comme vous le souhaitiez.

Je serai votre enseignant pour le cours Développement de produit média (DévProdMéd).

Le cours aura lieu (certains) lundis de 8:30 à 12:00 en salle T104 et vous trouverez toutes les informations relatives dans ce canal Teams.

J'utiliserai ce canal pour les échanges et les supports de cours.

Si vous avez des questions, n'hésitez pas à me contacter.

Je me réjouis de vous voir demain matin.
```

## Pendant le cours

Le fil rouge d'un cours est le suivant :

- Quiz du cours précédent animé par la personne qui enseigne
- Présentation animée par la personne qui enseigne
- Support de cours pratiqué par les personnes qui étudient
- Exercices pratiques réalisés par les personnes qui étudient
