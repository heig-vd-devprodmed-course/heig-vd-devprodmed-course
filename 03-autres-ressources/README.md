# Autres ressources

Ce dossier contient des ressources pour le cours _"Développement de produits
médias (DévProdMéd)"_.

## Utilisation de GitHub Classroom

[GitHub Classroom](https://classroom.github.com/) est utilisé pour gérer les
dépôts de cours.

1. Accéder à [GitHub Classroom](https://classroom.github.com/).
2. Créer une nouvelle organisation pour le cours avec l'année académique en
   cours (par exemple, `heig-vd-devprodmed-2025-2026`).
3. Changer les détails de l'organisation si nécessaire :
   - **Organization display name** : _"HEIG-VD DévProdMéd Course - GitHub
     Classroom (2025-2026)"_
   - **Description** : _"L'organisation GitHub Classroom pour le cours
     "Développement de produit média (DévProdMéd)" enseigné à la HEIG-VD."_
   - **URL**:
     <https://github.com/heig-vd-devprodmed-course/heig-vd-devprodmed-course>
4. Importer la liste des étudiant.es depuis le fichier CSV disponible sur
   [GAPS](https://gaps.heig-vd.ch/).
5. Créer un devoir (_"assignment"_) pour le mini-projet :
   1. **Assignment title**: _"DévProdMéd - Mini-projet"_
   2. **Custom repository prefix**: `devprodmed-mini-projet`
   3. **Deadline**: _none_
   4. **Individual or group assignment**: Individual assignment
   5. **Repository visibility**: Private
   6. **Grant students admin access to their repository**: Enabled
   7. **Add a supported editor**: _none_ (l'éditeur local sera utilisé)
   8. **Enable feedback pull requests**: Disabled (les étudiant.es soumettront
      des pull requests pour chaque étape du mini-projet)
6. Un lien vers le devoir sera généré et pourra être partagé avec les
   étudiant.es.
7. Chaque étudiant.e devra choisir sa personne dans la liste lors de l'accès au
   lien.
8. Un dépôt GitHub privé sera créé pour chaque étudiant.e, nommé
   `devprodmed-mini-projet-<github-username>`.
