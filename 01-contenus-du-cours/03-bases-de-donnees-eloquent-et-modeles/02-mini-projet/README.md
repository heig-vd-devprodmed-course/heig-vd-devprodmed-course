## Solution

La solution du mini-projet est accessible dans le dossier
`heig-vd-devprodmed-mini-projet/` du dépôt principal. Vous y trouverez :

- Les migrations pour les tables `posts` et `comments`.
- Les modèles `Post` et `Comment`.

>
> La solution est fournie à titre indicatif uniquement. Il est fortement
> recommandé de développer votre propre version avant de consulter la solution.
